# 教育短视频自动化生产流水线 (Edu Video Pipeline)

> **为30-50岁零基础学英语人群自动化生产教育短视频**

## 🎯 功能特性

- ✅ **智能教案生成**：针对零基础人群设计20-30秒知识点
- ✅ **AI视频生成**：使用即梦/Seedance生成温馨暖色调视频（调用 seedance skill 生成专业视频提示词）
- ✅ **封面图生成**：生活化场景 + 醒目中文标题
- ✅ **批量生产**：支持每日4个视频定时生产
- ✅ **自动推送**：通过Webhook发送到钉钉

## 🤖 执行步骤（MUST FOLLOW）

### Step 1: 生成教案

**Prompt Template:**
```
为30-50岁零基础/低起点学英语人群设计一个20-30秒的短视频教案。

主题：{用户输入主题}
类型：{生活口语/旅游口语/职场口语/英语习语/节日文化/固定搭配}

要求：
1. 中文为主要讲解语言，降低理解门槛
2. 英文内容简单实用，3-5个核心表达
3. 场景生活化，便于理解和记忆
4. 包含逐字稿（中英双语对照）

输出格式：
- 视频标题（中文，15字以内，吸引眼球）
- 教学知识点（3-5个核心表达）
- 场景描述（生活化、温馨、暖色调）
- 逐字稿（0-5秒、5-10秒、10-15秒、15-20秒、20-30秒）
- 中文讲解要点
- 英文示范发音
```

### Step 2: 生成视频提示词（调用 seedance skill）

本步骤会调用 `seedance` skill 生成专业的 Seedance 2.0 视频提示词。

**seedance skill 来源**: https://github.com/songguoxs/seedance-prompt-skill

**视频提示词模板（即梦/Seedance）:**
```
温馨教育短视频，{场景描述}，暖色调灯光。

人物：欧美年轻女性，20-28岁，金发或棕发，笑容亲切，穿着休闲舒适的日常服装（米色/浅蓝色毛衣或衬衫），给人专业但亲切的感觉。

场景：{具体场景} - 生活化、温馨、暖色调为主（米白/浅黄/暖橙色），避免冷冰冰的商务办公室风格。

镜头：
- 0-5秒：中景，模特微笑面对镜头，温馨背景
- 5-10秒：近景，模特讲解动作，表情生动
- 10-15秒：特写，展示口型或手势
- 15-20秒：中景，模特演示对话场景
- 20-30秒：全景，温馨结尾，出现知识点总结

风格：照片级真实感，柔光照明，浅景深，Instagram风格温馨生活照，16:9横屏。

氛围：温暖、鼓励、亲切、让学习者感到放松和自信。
```

### Step 3: 生成视频

**调用即梦AI服务：**

```bash
# 1. 检查服务状态
curl http://localhost:8000/ping

# 2. 生成视频（使用 jimeng-video-3.0-pro 模型）
curl -X POST http://localhost:8000/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer d56a24782ccca71a48f6449e67990550" \
  -d '{
    "model": "jimeng-video-3.0-pro",
    "messages": [{"role": "user", "content": "{视频提示词}, 30秒"}]
  }'
```

### Step 4: 生成封面图

**封面图提示词模板：**
```
教育短视频封面图，{主题相关的温馨生活场景}。

人物：与视频同一位欧美年轻女性模特，{具体穿着描述}，笑容亲切自然。

构图：人物占据画面上半部分，底部1/3留白用于标题文字。

背景：{生活化场景}，暖色调（米色/浅黄/橙色），柔和光线，虚化背景。

风格：Instagram风格生活方式摄影，照片级真实感，9:16竖屏适合手机观看。

氛围：温暖、亲切、专业但不严肃，让中老年学习者感到友好和可接近。

注意：画面底部留出足够空间放置标题文字，不要有过多装饰元素遮挡。
```

**标题设计规范：**
```
主标题（底部居中）：
- 文字：{视频标题}
- 字体：粗体中文，字号大
- 颜色：高对比度（如白色文字+深色阴影）
- 重点词汇用亮色突出（如橙色/黄色）

副标题（主标题下方）：
- 文字：20秒学会 / 零基础也能懂
- 字体：较小字号
- 颜色：柔和色调
```

## 💡 提示词最佳实践

### 人物一致性保证

每次生成时重复以下关键描述：
- **模特**：欧美年轻女性，20-28岁，金发或棕发
- **穿着**：米色/浅蓝色/浅粉色休闲服装
- **表情**：亲切微笑，专业但友好
- **风格**：生活化、非商务、温馨暖色调

### 场景设计原则

| ✅ 推荐场景 | ❌ 避免场景 |
|-----------|-----------|
| 温馨咖啡厅 | 冷冰冰的办公室 |
| 家庭客厅 | 正式的会议室 |
| 阳光公园 | 工业风格背景 |
| 舒适书房 | 科技感实验室 |
| 餐厅/厨房 | 医院/银行场景 |

### 中文标题技巧

- 使用数字："3句话学会..."
- 强调时间："20秒掌握..."
- 突出简单："零基础也能..."
- 制造好奇："原来老外这样说..."
- 解决痛点："出国旅游不再慌..."

## 📞 故障排除

### 服务未启动
```bash
cd /Users/xdf/jimeng-free-api-all
npm start
```

### Token 失效
1. 访问 https://jimeng.jianying.com/ 登录
2. F12 → Application → Cookies → 复制 `sessionid`
3. 更新 config.json 中的 token

### 积分不足
- 等待次日刷新（每日赠送66积分）
- 减少每日生产数量
- 使用多账号轮询

### 视频生成失败
- 检查提示词长度（不宜过长）
- 简化场景描述
- 尝试更换模型版本

# 视频生成提示词模板

## 通用人物设定（所有视频共用）

```
人物：欧美年轻女性，22-26岁，{金发/棕发}，笑容亲切自然，
穿着{米色/浅蓝色/浅粉色}休闲服装，给人专业但平易近人的感觉。
```

## 主题场景模板

### 1. 生活口语 - 餐厅场景

```
温馨教育短视频，餐厅点餐英语教学。

人物：欧美年轻女性，22岁，金发，笑容亲切，穿着浅米色针织衫配白色休闲裤。

场景：温馨西餐厅，暖黄色灯光，木质桌椅，窗外阳光，生活化布置，
暖色调为主（米白/浅黄/暖橙色），柔和自然光线，浅景深虚化背景。

镜头设计：
- 0-5秒：中景，模特微笑面对镜头，手持菜单，温馨餐厅背景
- 5-10秒：近景，模特讲解点餐用语，手势自然
- 10-15秒：特写，展示口型和发音
- 15-20秒：中景，模拟与服务员对话场景
- 20-30秒：全景，温馨结尾，展示美味食物

风格：照片级真实感，Instagram生活方式摄影，柔光照明，16:9横屏，暖色调。
氛围：温暖、亲切、让学习者感到放松，仿佛置身温馨餐厅。
```

### 封面标题设计规范

**标题结构：**
```
主标题格式：
"{场景痛点}？{数字}句{解决方案}{效果}"

示例：
- "餐厅点餐只会指菜单？3句万能英语轻松点餐"
- "打招呼只会说Hello？3句万能英语秒变自然"
- "机场迷路怎么办？5句英语搞定所有问题"
```

**高亮规则：**
| 元素 | 颜色 | 用途 |
|-----|------|------|
| 数字 | 橙色/金色 | 吸引眼球，强调数量 |
| 关键词 | 橙色 | 核心价值点 |
| 结果词 | 金色 | 最终效果 |

**示例配色：**
- "3句万能" → 橙色高亮
- "轻松点餐" → 金色强调
- "秒变自然" → 金色强调

## 镜头语言指南

### 景别使用

| 时间 | 景别 | 用途 |
|-----|------|------|
| 0-5秒 | 中景 | 建立场景，展示人物和环境 |
| 5-10秒 | 近景 | 讲解重点，展示表情 |
| 10-15秒 | 特写 | 强调细节，口型展示 |
| 15-20秒 | 中景 | 演示场景，动作展示 |
| 20-30秒 | 全景 | 总结收尾，营造氛围 |

### 光线要求

- **主光**: 柔和的暖色调光源
- **辅光**: 填充阴影，保持明亮
- **背景光**: 分离人物和背景
- **避免**: 冷色调、硬光、顶光

### 色彩调色板

```
主色调：
- 米白 #F5F5DC
- 浅黄 #FFF8DC
- 暖橙 #FF8C00
- 淡粉 #FFB6C1

辅助色：
- 浅蓝 #87CEEB
- 米色 #F5F5DC
- 暖灰 #D3D3D3

避免：
- 冷蓝色
- 工业灰色
- 高饱和度红色
```

# 视频生成提示词模板

## 通用人物设定（所有视频共用）

```
人物：欧美年轻女性，22-26岁，{金发/棕发}，笑容亲切自然，
穿着{米色/浅蓝色/浅粉色}休闲服装，给人专业但平易近人的感觉。
```

## 主题场景模板

### 1. 生活口语 - 餐厅场景

```
温馨教育短视频，餐厅点餐英语教学。

人物：欧美年轻女性，22岁，金发，笑容亲切，穿着浅米色针织衫配白色休闲裤。

场景：温馨西餐厅，暖黄色灯光，木质桌椅，窗外阳光，生活化布置，
暖色调为主（米白/浅黄/暖橙色），柔和自然光线，浅景深虚化背景。

镜头设计：
- 0-5秒：中景，模特微笑面对镜头，手持菜单，温馨餐厅背景
- 5-10秒：近景，模特讲解点餐用语，手势自然
- 10-15秒：特写，展示口型和发音
- 15-20秒：中景，模拟与服务员对话场景
- 20-30秒：全景，温馨结尾，展示美味食物

风格：照片级真实感，Instagram生活方式摄影，柔光照明，16:9横屏，暖色调。
氛围：温暖、亲切、让学习者感到放松，仿佛置身温馨餐厅。
```

## 封面图模板

### 通用封面结构

```
教育短视频封面图，{主题相关的温馨场景}。

人物：欧美年轻女性教师形象，22-26岁，{金发/棕发}，
亲切微笑，穿着{米色/浅蓝色/浅粉色}休闲服装，
自然放松的姿态，面向镜头。

构图：人物占据画面上半部分（约2/3），
底部1/3留白用于标题文字，中景拍摄。

背景：{与主题相关的生活化场景}，
暖色调（米色/浅黄/橙色/淡粉），
柔和光线，虚化处理，营造温暖亲切感。

风格：Instagram风格生活方式摄影，
照片级真实感，9:16竖屏比例，适合手机观看。

氛围：温暖、亲切、专业但不严肃，
让中老年学习者感到友好和可接近，吸引点击。

注意：画面底部留出足够空间放置大标题文字，
背景简洁不杂乱，人物形象清晰。
```

### 封面标题设计规范

```
视觉层次：
1. 主标题（大）：粗体中文，白色或浅色
   - 15字以内
   - 高对比度，带深色阴影
   - 重点词汇用亮色突出（#FF6B35 橙色）

2. 副标题（中）：较小字号
   - "20秒学会" / "零基础也能懂"
   - 柔和色调

3. 标签（小）：可选
   - "实用口语" / "旅游必备"
   - 圆角标签样式
```

## 镜头语言指南

### 景别使用

| 时间 | 景别 | 用途 |
|-----|------|------|
| 0-5秒 | 中景 | 建立场景，展示人物和环境 |
| 5-10秒 | 近景 | 讲解重点，展示表情 |
| 10-15秒 | 特写 | 强调细节，口型展示 |
| 15-20秒 | 中景 | 演示场景，动作展示 |
| 20-30秒 | 全景 | 总结收尾，营造氛围 |

### 光线要求

- **主光**: 柔和的暖色调光源
- **辅光**: 填充阴影，保持明亮
- **背景光**: 分离人物和背景
- **避免**: 冷色调、硬光、顶光

```
避免：

```
